4. html 기본 문서 만들기
    1) HTML
        -HyperTexMarkupEngine language
        -hypertext: 다른데있는 텍스쳐를 가져오는 - 문서를 서로 연결해주는 링크
        -웹문서 만드는 언어
    2) html 문서의 기본 구조
        -<!DOCTYPE html>
            -현재 문서가 html 언어로 작성한 웹문서하는 뜻
        -<html> ~</html>
            -웹문서의 시작과 끝나타내는 태그
        -<head> - </head>
            -웹브라우져가 웹문서를 해석하는데 필요한 저보를 입력하는 부분
        -<body>
            -실제로 웹부라우저 화면에 나타나는 내용임.
    3) 태그, 기억해 두면 좋은 4가지
        -태그는 소문자로 씀
        -여는 태그와 닫는 태그를 정확이 입력하기
        -적당하게 들여쓰기를 함
        -일부 태그는 속성과 함꼐 사용함

5. 시맨틱(semantic) 태그
    1)문서와 무관함
        -웹부라우져가 문서구조를 파악하는데 중요한 역할
    2)주요 시멘틱 태그
        -<header>
            -헤더 영역을 나타내는 태그
        - <nav>
            -내비게이션 영역을 나타내는 태그
        - <main>
            -핵심컨텐츠를 담는 태그
            -웹문서에서 한번만 사용할수있음
            -웹 문서마다 다르게 보여주는 내용으로 구성함
        -<article>
            -독립적인 콘텐츠를 담는 태그
            -심문 잡지
            -여러개 사용할수있고, 이안에 <section> 태그를 넣을수 있음
        -<section>
            -웹문서에서 콘텐츠 영역을 나타내는 태그
            -몇개의 콘테츠를 묶을떄사용함
        -<aside>
            -사이드바 영역을 나타내는 태그
        -<footer>
            -footer영역을 나타내는 태그
            -사이트 제작정보, 저작권정보, 연락처 등등 보통 이런정보 태그

7. 정리
    1) element == tag
    2) 속성(attribute) 과 값(value)
        -<태그이름 attribute="value">내용</>
        -태그기능 확장
    3) 부모 요소와 자식 요소
        -ex)
        <부모요소>
            <자식요소>
            </자식요소>
        </부모요소>            
    4) 빈태그 (empty tag)
        -종료태그가 없는 태그 지칭
        -<태그>
        -<태그/>
        -<태그 속성="속성값"/>
        -내용을 못넣기떄문에 대부분 속성값이 필요함
    5)head, body, html,
    <meta>
        -html 문서(웹페이지)에 관한 정보 (제작자, 내용, 키워드)를 검색엔진이나 브라우저에게 제공
        -decoding -변화 incoding

3. 표(table) 만들기
    1) 표의 구성 요소
        - 행 과 열 그리고 셀
    2)<table>
    3)행을 만드는 <tr> 셀을 만드는 td and th 
    4) 표의 구조를 지정하는 <thead>, <tbody>, <tfoot>
    -똑같은 컨셉 as head, body
    5)행이나 열을 합치는 rowspan, colspan 속성
        -<td rowspan ="합칠 셀의 개수"> 내용</td>
        -<td colspan ="합칠 셀의 개수"> 내용</td>
    6)열을 묶어주는 <col1>, <colgroup> 태그기능
        -<col> 태그는 열을 1개만 지정할 떄 사용
        -<colgroup> 태그는 <col> 태그를 2개 이상 묶어서 사용함

        -px
            -해당 픽셀만큼 표시함
