1. 연결 선택자
    1)하위 상위 
        -특정 요소를 기준으로 그안에 포함된 요소를
    2)자식요소
        =현재 요소를 기준으로 바로 한단계아래 요소를
    3)손자 요소
        -그자식요소의 한단계 아래 요소를
    4)형재 요소를   
        -부모 요소가 같은경우
        -먼저나오는 요소를 형요소, 나중에 요소를 동생요소

    
    4)하위 선택자 (descendant selector)
        -부모 요소에 포함된 하위요소를 모두 선택
        -자식 요소 뿐만 아니라 손자요소, 손자의 손자등모든 하윆
        -상위요소와 하위요소를 공백으로 구분함
        -기본형
        -상위요소 하위요소
        ex)
        section p {...}

    5)자식 선택자 (child selector)
        -하위 선택자와 다르게 자식요송에만 스타일을 적용
        -기본형
            -부모요소 > 자식요소
    
    6)인접 형재 선택자 (adjacent selector)
        -형제 요소중에서 첫번쨰 동생요소만 선택하는것
        -요소1 + 요소2
    
    7) 형제 선택자 (sibling selector){
        -모든 형제 요소에 적용됨
        -기본형
            -요소1 ~ 요소2
                h1 ~ p {
                    ~~~
                }
                h1의 형제인 모든 p 요소
    
2. 속성 선택자
    1) 특정 속성이 있느 요소를 선태하는 [속성] 선택자
    2) 대괄호를-[] 사이에 원하는 속성을 입력하면 됨
        - a[href] { ... }

    3)특정속성값이 있는 요소를 선택하는 [속성=속성값] 선택자
        -a[target = _blank]
            -a 요소중에서 target속성값이 _blank인 것만 선택함

    4) 여러 값 중에서 특정 속성값이 포함된 속성 요소를 선택하는 [속석~=값] 선택자
    
    5) 특정속성값이 포함된 요소를 선택하는 [속성|=값] 선택자
        -속성값은 한단어로 일치해야함.
    6)^=
    
    6)$=

    7)*=
