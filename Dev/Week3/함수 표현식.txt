화살표 함수
    -ES6 버전부터 => 표기법 사용함
    함수 선언을 좀더 간단하게 적성가능
    기본형
        - (매개변수) => {함수 내용}
    매개변수가 없을 경우
        -예)
            const h1 = function(){
                return "안녕하세요"
            }

        -화살표 연산자
            const h1 = () => {return "안녕하세요";};

        -더 축약시키면
            const h1 = () => "안녕하세요";
