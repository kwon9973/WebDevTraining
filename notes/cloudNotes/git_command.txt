1. Git의 기초 명령어
    1) git 버젼 확인
        -git --version // --옵션을 줄떄 씀
    2) 현재 git 설정 확인
        -git config --list
    3) 사용자 정보가 없다면 설정 
        -git config --global user.name
        git config --global user.email kwon9973@gmail.com
